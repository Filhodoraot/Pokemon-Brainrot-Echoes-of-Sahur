# Brainrot movepools V1

This file defines movepool identity for the first playable build.

Important:

- Use existing FireRed moves first.
- Custom move names come later.
- Not every Brainrot evolves.
- Move identity should follow the Brainrot's role, not just old Pokemon logic.

## Move style tags

Each Brainrot can have one or more move style tags.

| Tag | Meaning |
|---|---|
| BRUTE | physical pressure |
| SOUND | Sahur/noise influence |
| MEME | confusion/weird utility |
| SPEED | fast attacks |
| TRICK | annoying, stealing, accuracy, status |
| MEMORY | mind/lore attacks |
| WATER | water rhythm moves |
| FIRE | fire/espresso aggression |
| PLANT | grass/banana/nature moves |

## Development order for movepools

1. Use existing moves in learnsets.
2. Balance early game.
3. Rename existing moves if safe.
4. Add custom move names.
5. Add custom effects.
6. Add signature moves.
7. Add legendary mechanics.
