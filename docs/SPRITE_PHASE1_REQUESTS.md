# Sprite Phase 1 Requests

This file is the first sprite checklist for **Pokemon Brainrot: Echoes of Sahur**.

Goal: stop the playtest from feeling like FireRed with renamed Pokemon.

## Style rules

Use a GBA-friendly pixel style:

- transparent background
- readable silhouette
- maximum 16 colors per sprite when possible
- strong black/dark outline
- front sprite and back sprite should clearly be the same creature
- adult/final forms should look more aggressive or dangerous than baby forms
- keep each design simple enough to read at small size

## Required files per Brainrot

For each Brainrot, make:

1. front battle sprite
2. back battle sprite
3. party/menu icon
4. palette

Suggested working sizes before conversion:

- front sprite: 64x64
- back sprite: 64x64
- icon: 32x32

## Protagonist first

### ROAN

Roan should not look like Red.

Design:

- no cap
- dark visible hair
- dark hoodie or jacket
- dark pants
- small blue or orange detail
- explorer/researcher street vibe
- serious expression

Needed sprites:

- overworld walking sheet
- battle back sprite
- trainer/card sprite
- intro/name select sprite later

## Rival first

### KIUNIN

Design:

- spoiled, loud, anxious, competitive
- more flashy than Roan
- brighter jacket or hair detail
- aggressive pose
- looks like he wants attention

Needed sprites:

- overworld walking sheet
- trainer battle sprite
- optional back sprite later
