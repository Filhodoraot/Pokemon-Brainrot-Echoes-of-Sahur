- 100 Brainrots total.
- 30 Brainrots have evolution/forms.
- 70 Brainrots are single-stage.
- Starters always have evolution.

For sprite work:

- `EVOLVES` = needs more than one sprite/form.
- `SINGLE` = only needs one sprite for now.

## Notes for sprite artist

Priority order:

1. Draw the 3 starters first.
2. Draw early route single sprites next: Noobini, Tim Cheese, Pipi Kiwi, Raccooni.
3. Draw story legends/forms later: 67, Tung Tung, Strawberry Helefanto, Skibidy.
4. Do not make all 100 evolve. Only the 30 listed above need multiple forms.

## Important design note

A Brainrot can have a lore stage like Baby/Teen/Adult even if it does not evolve in gameplay.

Example:

- Tim Cheese can be stage Teen, but no evolution.
- Raccooni can be stage Adult, but no evolution.
