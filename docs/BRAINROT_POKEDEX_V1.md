## Temporary implementation rule

For the first playable build, do not add 100 new species yet.

Instead:
1. Replace existing early Pokemon names and data.
2. Keep old sprites as temporary placeholders.
3. Make the first hour playable.
4. Add new species and sprites after the base works.
