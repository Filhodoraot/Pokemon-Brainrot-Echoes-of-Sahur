# Brainrot evolution sprite list V2

Correction:

Legendary Brainrots do not evolve.

They can have boss/cutscene forms later, but they are not part of normal Baby -> Teen -> Adult evolution.

Rule:

- 100 Brainrots total.
- 30 non-legendary Brainrots have evolution/forms.
- 70 Brainrots are single-stage.
- Starters always evolve.
- Legendaries do not evolve.

For sprite work:

- `EVOLVES` = needs Baby/Teen/Adult or normal evolution sprites.
- `SINGLE` = only needs one gameplay sprite for now.
- `LEGENDARY` = one main sprite for now, special boss forms later only if requested.
