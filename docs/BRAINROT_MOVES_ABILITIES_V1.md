## Legendary signature moves

### 67 / Six Seven

Signature move: Six Seven Loop.

Effect idea:
- Target repeats its last move or loses control.
- High chance to confuse.
- Represents meaningless repetition.

### Skibidy Toilet

Signature move: Toilet Rift.

Effect idea:
- Strong glitch damage.
- Changes the battlefield flavor.
- May confuse or lower accuracy.

### Strawberry Helefanto

Signature move: Red Memory.

Effect idea:
- Restores HP.
- Clears confusion.
- Can reveal hidden truth in story events.

### Tung Tung Sahur God Form

Signature move: God Beat.

Effect idea:
- Strong sound/memory damage.
- Lowers target special defense.
- Chance to disable the last move.

### Supreme Tung Tung

Signature move: Supreme Beat.

Effect idea:
- Final boss move.
- Strong damage.
- Hits memory and sound.
- May lower multiple stats.

## Legendary abilities

### 67: Loop Logic

Effect idea:
- Once per battle, when hit, 67 forces the attacker to repeat the same move next turn.
- Fits the loop theme.

### Skibidy Toilet: Reality Clog

Effect idea:
- On entry, lowers enemy accuracy or speed.
- Represents corrupted space.

### Strawberry Helefanto: Memory Bloom

Effect idea:
- Heals status of allies or protects from confusion.
- Story role: protects memories erased by Sahur noise.

### Tung Tung God Form: Cognitive Theft

Effect idea:
- Each turn lowers one mental stat from the opponent.
- Boss-only ability at first.

### Supreme Tung Tung: Stolen IQ

Effect idea:
- Gains strength as the battle continues.
- Could boost special attack or all stats after certain HP thresholds.
- Final boss only.

## Implementation rule

For the first playable version, do not code all abilities immediately.

Safer path:
1. Rename existing moves first.
2. Reuse existing effects.
3. Add custom effects later.
4. Add legendary custom abilities last.

This keeps the build from exploding like a Svinina Bombardino.
