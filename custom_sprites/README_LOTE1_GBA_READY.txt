# Lote 1 convertido para GBA

Este pacote foi gerado a partir de `brainrot_sprites_lote_001_034_corrigido_v4.zip`.

Coloque a pasta `custom_sprites/` na raiz do repo `Pokemon-Brainrot-Echoes-of-Sahur`.

O build já tem `tools/apply_custom_sprite_dropins.py`, então esses arquivos serão copiados para os slots certos.

## Ainda ficou fora

Os outros Brainrots do lote 001-034 ainda precisam de slots/species no jogo antes de entrar.
Exemplos: Tung Tung Tung Sahur, Bandito Bobritto, Boneca Ambalabu, Cacto Hipopotamo, etc.

## Observações

- front/back foram redimensionados para 64x64.
- icon foi convertido para 32x64 com dois frames iguais.
- normal.pal e shiny.pal foram gerados como paletas de 16 cores.
- Isso é uma conversão técnica rápida. Depois dá para polir manualmente os sprites que ficarem pequenos ou muito detalhados.
